# 📄 Relatório de Implementação de Serviços AWS — Abstergo Industries

> Projeto desenvolvido como parte de estudo prático em computação em nuvem com foco em **redução de custos utilizando serviços AWS**.

---

## 🏢 Sobre o Projeto

Este repositório contém um relatório técnico de implementação de serviços AWS na empresa fictícia **Abstergo Industries**, do setor farmacêutico. O objetivo foi identificar e propor 3 serviços da Amazon Web Services capazes de gerar **redução de custos imediata** na infraestrutura de TI da empresa.

---

## ☁️ Serviços AWS Implementados

| Etapa | Serviço | Foco |
|-------|---------|------|
| 1 | **Amazon S3** | Armazenamento em nuvem substituindo servidores físicos |
| 2 | **AWS Lambda** | Computação serverless eliminando servidores ociosos |
| 3 | **Amazon RDS (Reserved Instances)** | Banco de dados gerenciado com até 72% de desconto |

---

## 💰 Resultados Esperados

- Redução de custos de infraestrutura entre **50% e 70%** no primeiro ano
- Eliminação de gastos com hardware físico e manutenção manual
- Maior disponibilidade, escalabilidade e segurança dos sistemas
- Conformidade com requisitos regulatórios do setor farmacêutico (ANVISA, FDA)
